# Проблема с генерацией изображений через Kandinsky

## Проблема:
Kandinsky API возвращает 404 (Not Found)

## Причины:
1. **Неправильный endpoint** - API endpoint может быть другим
2. **Недоступно в тарифе** - Генерация изображений может быть недоступна в вашем тарифе Yandex Cloud
3. **Нужны дополнительные разрешения** - API ключ может не иметь нужных разрешений

## Решения:

### Вариант 1: Использовать оригинальное изображение или Pexels
- Оригинальное изображение из статьи работает
- Pexels требует API ключ (бесплатно на https://www.pexels.com/api/)
- Unsplash работает без ключа (но может быть медленным)

### Вариант 2: Настроить Pexels API (рекомендуется)
1. Зарегистрируйтесь на https://www.pexels.com/api/
2. Получите API ключ
3. Добавьте в `Backend/BOT_TOKEN.env`:
   ```env
   PEXELS_API_KEY=ваш_ключ
   ```
4. Перезапустите бэкенд

### Вариант 3: Проверить доступность Kandinsky в Yandex Cloud
1. Откройте консоль Yandex Cloud
2. Проверьте, доступна ли генерация изображений в вашем тарифе
3. Проверьте документацию API для правильного endpoint

## Текущий статус:
- ✅ Оригинальное изображение - работает
- ⚠️ Pexels - требует API ключ
- ⚠️ Unsplash - работает, но может быть медленным
- ❌ Kandinsky - недоступен (404 ошибка)

## Рекомендация:
Используйте оригинальное изображение или настройте Pexels API для получения качественных изображений.
