# Настройка домена для Telegram Login Widget

## ⚠️ Важно: BotFather не принимает localhost

Telegram Login Widget **не работает с localhost** напрямую. Нужно использовать один из вариантов ниже.

## Вариант 1: Использование ngrok (рекомендуется для разработки)

### Шаг 1: Установите ngrok
1. Скачайте с https://ngrok.com/download
2. Или установите через npm: `npm install -g ngrok`

### Шаг 2: Запустите туннель
```bash
ngrok http 3000
```

### Шаг 3: Скопируйте домен
Вы получите URL вида: `https://abc123.ngrok.io`

### Шаг 4: Укажите домен BotFather
Отправьте BotFather только домен (без https://):
```
abc123.ngrok.io
```

## Вариант 2: Использование реального домена

Если у вас есть домен, укажите его:
```
yourdomain.com
```

## Вариант 3: Временное решение для тестирования

Можно временно использовать любой бесплатный хостинг (например, Vercel, Netlify) для тестирования виджета.

## Важно:

1. **Указывайте БЕЗ протокола** (не `https://abc123.ngrok.io`, а просто `abc123.ngrok.io`)
2. **БЕЗ порта** (не `yourdomain.com:3000`, а просто `yourdomain.com`)
3. После указания домена BotFather подтвердит настройку

## Пример ответа BotFather:

После указания домена вы увидите:
```
Success! Domain updated. The widget will be available on: abc123.ngrok.io
```

## После настройки:

1. Подождите 1-2 минуты для применения изменений
2. Откройте сайт через ngrok URL (не через localhost!)
3. Обновите страницу сайта (Ctrl+F5)
4. Откройте меню авторизации
5. Виджет должен загрузиться без ошибки "Username invalid"
