# Настройка Query-параметров и Тела запроса

## Query-параметры (Query-параметры)

**Оставьте пустым** - для Yandex Cloud AI Assistant API query-параметры не нужны.

Или удалите все параметры, если они есть (кнопка корзины).

## Тело запроса (Тело запроса)

В поле **"Тело запроса"** введите JSON структуру:

```json
{
  "prompt": {
    "id": "{{prompt_id}}"
  },
  "input": "{{input_text}}"
}
```

**Важно:** 
- `{{prompt_id}}` - это переменная, которая будет заменена на значение параметра `prompt_id` из "Параметры инструмента"
- `{{input_text}}` - это переменная, которая будет заменена на значение параметра `input_text` из "Параметры инструмента"

### Альтернативный вариант (если переменные не работают):

Если интерфейс не поддерживает переменные в формате `{{variable}}`, попробуйте:

```json
{
  "prompt": {
    "id": "fvtfdp5dm8r044bnumjl"
  },
  "input": ""
}
```

Или используйте формат, который поддерживает ваш интерфейс для подстановки значений параметров.

### Если нужно добавить стиль:

Если вы добавили параметр `style` в "Параметры инструмента", можно включить его в тело запроса:

```json
{
  "prompt": {
    "id": "{{prompt_id}}"
  },
  "input": "Стиль: {{style}}\n\nТекст статьи:\n{{input_text}}"
}
```

## Проверка формата

Убедитесь, что:
1. JSON валидный (можно проверить на jsonlint.com)
2. Используются правильные имена полей (`prompt`, `input`)
3. Структура соответствует документации Yandex Cloud AI Assistant API

## Если формат отличается

Если Yandex Cloud использует другой формат запроса, проверьте документацию API и адаптируйте тело запроса соответственно.
