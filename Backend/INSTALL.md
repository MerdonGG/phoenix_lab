# Установка зависимостей для бэкенда

## Минимальная установка (для авторизации через бота)

Для работы авторизации через бота достаточно установить только базовые зависимости:

```bash
pip install flask flask-cors python-dotenv
```

Или используйте файл без aiogram:

```bash
pip install -r requirements-backend.txt
```

## Полная установка (включая отправку статей)

Если нужна функция отправки статей в Telegram каналы, установите все зависимости:

```bash
pip install -r requirements.txt
```

**Примечание:** `aiogram` требует компиляции с Rust. Если возникают проблемы:

1. Установите Rust: https://rustup.rs/
2. Или используйте предкомпилированные wheels (если доступны для вашей версии Python)

## Проверка установки

После установки запустите бэкенд:

```bash
python server.py
```

Если aiogram не установлен, вы увидите предупреждение, но авторизация будет работать.
