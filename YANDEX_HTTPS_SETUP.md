# Настройка HTTPS-запроса в Yandex Cloud

## Заполнение полей:

### 1. **Имя инструмента** (обязательное поле):
```
Рерайт статьи через AI
```
или
```
Article Rewrite Tool
```

### 2. **Инструкция для агента**:
```
Используй этот инструмент для переписывания статей в разных стилях. 
Принимай текст статьи и стиль рерайта, возвращай переписанный текст.
```

### 3. **URL** (обязательное поле):
```
https://rest-assistant.api.cloud.yandex.net/v1/responses
```

**Важно:** Это базовый URL API Yandex Cloud AI Assistant. Если в вашем проекте используется другой endpoint, уточните его в документации.

### 4. **Метод**:
Выберите **POST** (не GET!)

### 5. **Дополнительные параметры** (если есть):
- **Headers (Заголовки):**
  - `Authorization: Bearer {YANDEX_CLOUD_API_KEY}`
  - `Content-Type: application/json`

### 6. **Параметры инструмента** (в разделе "Форма"):

Добавьте следующие параметры:

**Параметр 1:**
- **Имя:** `prompt_id`
- **Тип:** Строка
- **Описание:** ID промпта ассистента (например: fvtfdp5dm8r044bnumjl)

**Параметр 2:**
- **Имя:** `input_text`
- **Тип:** Строка  
- **Описание:** Текст статьи для обработки

**Параметр 3:**
- **Имя:** `style`
- **Тип:** Строка
- **Описание:** Стиль рерайта (scientific, meme, casual)

## Альтернативный вариант (если используется другой формат API):

Если Yandex Cloud использует другой формат запроса, проверьте документацию API. 

**Структура запроса может быть такой:**
```json
{
  "prompt": {
    "id": "fvtfdp5dm8r044bnumjl"
  },
  "input": "Текст для обработки"
}
```

## Важно:

⚠️ **Обратите внимание:** В коде приложения мы используем Python SDK (`openai` библиотеку), которая автоматически формирует запросы. Если вы настраиваете HTTPS-запрос через интерфейс Yandex Cloud, убедитесь, что формат запроса совпадает с тем, что ожидает API.

## Проверка:

После настройки:
1. Сохраните инструмент
2. Получите ID инструмента
3. Добавьте его в `YANDEX_CLOUD_ASSISTANT_ID` в файле `Backend/BOT_TOKEN.env`
