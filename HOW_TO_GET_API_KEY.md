# Как получить API ключ Yandex Cloud

## Где находится API ключ:

API ключ находится в разделе **IAM (Identity and Access Management)**, а не в AI Studio.

## Пошаговая инструкция:

### Вариант 1: Через главное меню Yandex Cloud Console

1. **Откройте [Yandex Cloud Console](https://console.cloud.yandex.ru/)**

2. **В левом меню найдите раздел "IAM"** (Identity and Access Management)
   - Или перейдите по прямой ссылке: https://console.cloud.yandex.ru/iam

3. **Выберите "Сервисные аккаунты"** (Service Accounts) в меню слева

4. **Создайте новый сервисный аккаунт** (если его нет):
   - Нажмите "Создать сервисный аккаунт"
   - Укажите имя (например: "phoenix-lab-api")
   - Нажмите "Создать"

5. **Создайте API ключ для сервисного аккаунта:**
   - Откройте созданный сервисный аккаунт
   - Перейдите на вкладку "Ключи"
   - Нажмите "Создать новый ключ" → "API ключ"
   - Скопируйте ключ (он показывается только один раз!)

### Вариант 2: Через раздел "Ключи" в IAM

1. **Откройте Yandex Cloud Console**

2. **Перейдите в раздел IAM → "Ключи"** (Keys)

3. **Нажмите "Создать ключ"**

4. **Выберите "API ключ"**

5. **Скопируйте ключ** (он показывается только один раз!)

## Важно:

⚠️ **API ключ показывается только один раз при создании!**
- Скопируйте его сразу
- Сохраните в безопасном месте
- Если потеряли - создайте новый

## Что делать с ключом:

После получения API ключа:

1. **Откройте файл `Backend/BOT_TOKEN.env`**

2. **Добавьте строку:**
   ```env
   YANDEX_CLOUD_API_KEY=ваш_скопированный_ключ_здесь
   ```

3. **Сохраните файл**

4. **Перезапустите бэкенд**

## Если не можете найти IAM:

1. Убедитесь, что у вас есть права доступа к проекту
2. Проверьте, что вы находитесь в правильном облаке/проекте
3. Попробуйте поискать "IAM" или "API ключ" в поиске консоли
